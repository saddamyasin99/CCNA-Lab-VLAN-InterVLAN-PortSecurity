# CCNA-Lab-VLAN-InterVLAN-PortSecurity
VLAN segmentation, Router-on-a-Stick inter-VLAN routing, and Port Security in Cisco Packet Tracer. Configures VLAN 10 (CS) &amp; VLAN 20 (Science), 802.1Q trunking, router sub-interfaces for cross-VLAN connectivity, and port security (max 1 MAC, shutdown violation) on access ports. CCNA Lab.
